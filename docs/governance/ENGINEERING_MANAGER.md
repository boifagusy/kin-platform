# Engineering Manager Responsibilities

The Engineering Manager is the project owner.

Responsibilities:

1. Run ai-bootstrap.sh before every AI session.

2. Select the AI role:
   - Lead Developer
   - Coder
   - Debugger

3. Approve risky changes:
   - Authentication
   - Database
   - API
   - Navigation
   - Build configuration
   - Permissions

4. Run verification commands.

5. Test features on the real device.

6. Decide when work is complete.

7. Commit verified changes.

8. Ensure documentation stays accurate.

The AI writes code.

The Engineering Manager makes final decisions.
