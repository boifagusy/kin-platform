# KIN COPY-PASTE ROUTING GUIDE

Always begin every AI session by running:

./scripts/ai-bootstrap.sh

Paste the entire output into the AI before asking any question.

The bootstrap already provides:

- git status
- git diff
- git log
- open sessions
- project status
- recent commits

There is no SESSION_STATE.md.

Current state is derived from:

- git
- AGENT_LOG.md
- PROJECT_STATUS.md

Never create additional state documents.
